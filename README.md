# Blinkit Sales Analysis Dashboard

## Project Overview
This project analyzes Blinkit sales data using SQL and Power BI to uncover key business insights.

## Tools Used
- SQL (Data Cleaning & Analysis)
- Power BI (Interactive Dashboard)
- Excel / CSV (Data Source)

## Key Features
- KPI cards for Total Sales, Avg Sales, Items, Ratings
- Interactive filters (Outlet Size, Location, Item Type)
- Sales trend and category analysis

![Dashboard](dashboard.png)

## Key Insights
- Tier 3 outlets generate highest sales
- Medium outlet size performs best
- Supermarket Type1 leads in revenue
